# 1주차 프리코스 - 숫자야구게임

## 🚀 기능 목록
1. [ ] 랜덤 번호를 생성한다
2. [ ] 사용자에게 입력을 받는다


< 2번 - 예외 처리 >

  - [ ] 입력값이 3자리 수가 아닌 경우
        
  - [ ] 숫자가 중복될 경우
        
  - [ ] 공백이 포함될 경우
        
  - [ ] 숫자가 아닌 문자를 입력한 경우
        
3. [ ] Ball 개수를 구한다
4. [ ] Strike 개수를 구한다
5. [ ] 사용자에게 Ball, Strike 개수를 보여준다.
6. [ ] 사용자가 숫자를 맞췄을 때 게임을 시작/종료 여부를 사용자에게 입력받는다
7. [ ] 입력값이 1이면 게임을 새로 시작한다.
8. [ ] 입력값이 2이면 게임을 종료한다.
9. [ ] 입력값이 1과 2가 아니면 게임 시작/종료 여부를 사용자에게 다시 입력받는다.
